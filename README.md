# Personal CV Website

This repository contains the source code for my **personal CV website**, built to showcase my skills, projects, and professional background as I develop my career in software development.

The website demonstrates my understanding of **HTML, CSS, JavaScript, and responsive web design using Bootstrap**.

---

## Live Website

You can view the live version of the website here:

**https://dmjnr.github.io/cv-website/**

---

## Project Purpose

The purpose of this project is to:

* Present my **professional profile and technical skills**
* Showcase **projects I have developed**
* Provide an easy way for recruiters or collaborators to **contact me**
* Demonstrate my understanding of **frontend development fundamentals**

---

## Technologies Used

* **HTML5** – page structure and semantic layout
* **CSS3** – styling and layout customisation
* **JavaScript** – interactive elements
* **Bootstrap** – responsive design and layout components

---

## Features

* Responsive layout for **desktop, tablet, and mobile devices**
* Portfolio section displaying development projects
* Interactive elements powered by JavaScript
* Contact section for professional enquiries
* Downloadable CV

---

## Projects Highlighted
